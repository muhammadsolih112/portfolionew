# Portfolio

A modern, animated, and fully responsive portfolio website for Muhammad Solih with an admin panel and video edit system.

## Features

- 🌍 Multi-language support (English, Uzbek, Russian)
- 🎥 Video upload and management system
- 💬 Comment system with pinning functionality
- 📱 Fully responsive design
- ✨ Smooth animations with Framer Motion
- 🎨 Modern UI with Tailwind CSS

## Technologies

- React
- Framer Motion
- Tailwind CSS
- Vite

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
